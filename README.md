# Air-Quality-Monitioring
Air Quality Monitoring System using Thingspeak | ESP8266 | Blynk IOT Projects.

In this experiment we can see how to monitor the Air Quality using MQ135 Sensor and Thingspeak.

# Required Components
1. ESP8266
2. MQ135 Sensor
3. DHT11 Sensor
4. Connecting Cable
